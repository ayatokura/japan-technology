効果的な質問の仕方

質問に含めるべき情報

  1. Terraformのバージョン
terraform version
  1. プロバイダーのバージョン
terraform {
  required_providers {
    aws = {
      source  = "hashicorp/aws"
      version = "~> 5.0"
    }
  }
}
  1. エラーメッセージ
Error: creating EC2 Instance: UnauthorizedOperation: You are not authorized
to perform this operation.
  1. 関連する設定ファイル
resource "aws_instance" "web" {
  ami           = "ami-xxxxx"
  instance_type = "t2.micro"
}
  1. 実行したコマンド
terraform init
terraform plan
terraform apply
  1. 期待される動作と実際の動作
  • 期待: EC2インスタンスが作成される
  • 実際: 認証エラーが発生

質問のテンプレート

## 環境
- Terraform バージョン: 1.7.0
- プロバイダー: AWS 5.31.0
- OS: macOS 14.0

## 問題の説明
EC2インスタンスを作成しようとすると、認証エラーが発生します。

## 設定ファイル
```hcl
resource "aws_instance" "web" {
  ami           = "ami-xxxxx"
  instance_type = "t2.micro"
}

エラーメッセージ

Error: creating EC2 Instance: UnauthorizedOperation

試したこと

  1. AWS認証情報を確認
  2. IAMポリシーを確認

期待される動作

EC2インスタンスが正常に作成される


## トラブルシューティングのチェックリスト

質問する前に、以下を確認してください。

- [ ] 公式ドキュメントを確認した
- [ ] エラーメッセージで検索した
- [ ] `terraform validate`を実行した
- [ ] `terraform fmt`を実行した
- [ ] ログレベルをDEBUGに設定して確認した
- [ ] 既存のIssueやフォーラムの投稿を検索した

## セキュリティに関する報告

セキュリティの脆弱性を発見した場合は、公開せずに報告してください。

**報告先**: [https://www.hashicorp.com/security](https://www.hashicorp.com/security)

**注意事項**:
- 公開のIssueやフォーラムには投稿しない
- 詳細な再現手順を含める
- 影響範囲を明記する
