CLIのインストール

Terraform CLIは、Terraformを実行するためのコマンドラインツールです。このページでは、各オペレーティングシステムでのTerraform CLIのインストール方法を説明します。
最終更新: 2026/2/3

CLIのインストール

Terraform CLIは、Terraformを実行するためのコマンドラインツールです。このページでは、各オペレーティングシステムでのTerraform CLIのインストール方法を説明します。

インストール前の確認

システム要件

Terraformは以下のオペレーティングシステムで動作します。

  • macOS (Intel / Apple Silicon)
  • Linux (x86_64 / ARM64)
  • Windows (x86_64 / ARM64)
  • FreeBSD
  • OpenBSD
  • Solaris

必要な権限

インストールには、システムにソフトウェアをインストールする権限が必要です。

インストール方法

macOS

macOSでは、Homebrewを使用したインストールが推奨されます。

Homebrewを使用する方法（推奨）

# Homebrewのインストール（未インストールの場合）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Terraformのインストール
brew tap hashicorp/tap
brew install hashicorp/tap/terraform

手動インストール

  1. Terraform公式ダウンロードページにアクセス
  2. macOS用のバイナリをダウンロード
  3. ZIPファイルを解凍
  4. terraformバイナリを/usr/local/bin/に移動
# ダウンロードしたZIPファイルを解凍
unzip terraform_*_darwin_amd64.zip

# バイナリを移動
sudo mv terraform /usr/local/bin/

# 実行権限を付与
sudo chmod +x /usr/local/bin/terraform

Linux

Linuxでは、パッケージマネージャーまたは手動インストールが可能です。

Ubuntu/Debian

# 必要なパッケージのインストール
sudo apt-get update && sudo apt-get install -y gnupg software-properties-common

# HashiCorpのGPGキーを追加
wget -O- https://apt.releases.hashicorp.com/gpg | \
gpg --dearmor | \
sudo tee /usr/share/keyrings/hashicorp-archive-keyring.gpg

# HashiCorpのリポジトリを追加
echo "deb [signed-by=/usr/share/keyrings/hashicorp-archive-keyring.gpg] \
https://apt.releases.hashicorp.com $(lsb_release -cs) main" | \
sudo tee /etc/apt/sources.list.d/hashicorp.list

# パッケージリストを更新してTerraformをインストール
sudo apt update
sudo apt-get install terraform

CentOS/RHEL/Fedora

# HashiCorpのリポジトリを追加
sudo yum install -y yum-utils
sudo yum-config-manager --add-repo https://rpm.releases.hashicorp.com/RHEL/hashicorp.repo

# Terraformのインストール
sudo yum -y install terraform

手動インストール（全Linuxディストリビューション）

# 最新版のダウンロード（バージョンは適宜変更）
wget https://releases.hashicorp.com/terraform/1.7.0/terraform_1.7.0_linux_amd64.zip

# 解凍
unzip terraform_1.7.0_linux_amd64.zip

# バイナリを移動
sudo mv terraform /usr/local/bin/

# 実行権限を付与
sudo chmod +x /usr/local/bin/terraform

Windows

Windowsでは、Chocolateyを使用したインストールまたは手動インストールが可能です。

Chocolateyを使用する方法（推奨）

# Chocolateyのインストール（未インストールの場合）
Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))

# Terraformのインストール
choco install terraform

手動インストール

  1. Terraform公式ダウンロードページにアクセス
  2. Windows用のバイナリをダウンロード
  3. ZIPファイルを解凍
  4. terraform.exeを任意のディレクトリに配置（例: C:\terraform）
  5. システム環境変数のPATHに追加

環境変数の設定:

  1. 「システムのプロパティ」→「環境変数」を開く
  2. 「システム環境変数」の「Path」を編集
  3. Terraformのバイナリがあるディレクトリを追加（例: C:\terraform）

インストールの確認

インストールが完了したら、以下のコマンドでバージョンを確認します。

terraform version

出力例:

Terraform v1.7.0
on darwin_arm64

正しくバージョンが表示されれば、インストールは成功です。

Terraformのアップグレード

Homebrewを使用している場合（macOS）

brew update
brew upgrade hashicorp/tap/terraform

パッケージマネージャーを使用している場合（Linux）

# Ubuntu/Debian
sudo apt-get update
sudo apt-get upgrade terraform

# CentOS/RHEL/Fedora
sudo yum update terraform

Chocolateyを使用している場合（Windows）

choco upgrade terraform

手動インストールの場合

新しいバージョンをダウンロードして、既存のバイナリを置き換えます。

バージョン管理ツール（tfenv）

複数のTerraformバージョンを管理したい場合は、tfenvの使用を検討してください。

tfenvのインストール（macOS/Linux）

# Homebrewを使用（macOS）
brew install tfenv

# Gitから直接インストール（Linux）
git clone https://github.com/tfutils/tfenv.git ~/.tfenv
echo 'export PATH="$HOME/.tfenv/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc

tfenvの使用方法

# 利用可能なバージョンの一覧
tfenv list-remote

# 特定のバージョンをインストール
tfenv install 1.7.0

# 使用するバージョンを指定
tfenv use 1.7.0

# インストール済みバージョンの確認
tfenv list

エディタとIDE

Terraformのコードを効率的に記述するために、エディタやIDEのプラグインを使用することをお勧めします。

Visual Studio Code

推奨プラグイン:

  • HashiCorp Terraform（公式）
    • シンタックスハイライト
    • オートコンプリート
    • フォーマット機能
    • リンティング

インストール方法:

  1. VS Codeを開く
  2. 拡張機能タブを開く
  3. "HashiCorp Terraform"を検索してインストール

IntelliJ IDEA / PyCharm

推奨プラグイン:

  • Terraform and HCL

その他のエディタ

  • Vim: vim-terraform
  • Emacs: terraform-mode
  • Sublime Text: Terraform

トラブルシューティング

コマンドが見つからない

症状: terraform: command not found

解決方法:

  1. PATHが正しく設定されているか確認
  2. ターミナルを再起動
  3. インストールディレクトリを確認
# PATHの確認
echo $PATH

# Terraformの場所を確認
which terraform

権限エラー

症状: Permission denied

解決方法:

# 実行権限を付与
sudo chmod +x /usr/local/bin/terraform

バージョンの不一致

異なるプロジェクトで異なるTerraformバージョンが必要な場合は、tfenvの使用を検討してください。
